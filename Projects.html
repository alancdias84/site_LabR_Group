Projects

Open-source projects built by the LabR Group — statistical tools for clinical laboratory quality management, all freely available on GitHub.

LabRI Packed

A comprehensive Shiny application for computing reference intervals using robust statistical methods, partition testing, and IFCC-compliant reporting.

LabRI Chatbot

An AI-powered chatbot that answers questions about reference intervals, guiding lab professionals through method selection and interpretation.

SQC Dashboard

Interactive Statistical Quality Control dashboard implementing Westgard rules, sigma metrics and OPSpecs charts in a Shiny application.

Method Validation Tools

R-based workflows for bias, precision, linearity and measurement uncertainty following CLSI EP protocols.

Delta Check ML

Machine learning models for automated delta check flagging — detecting transcription errors and sample mix-ups in LIS data.

Open Book

A freely available online book covering statistical quality management in clinical laboratories using R. Written in R Markdown / Quarto.
